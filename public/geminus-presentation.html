Geminus

Your Digital Twin Platform

A unified platform for managing, visualizing and understanding buildings — with 3D, 360°, AI and IoT.

The Problem

🗄️

Fragmented data

BIM, operations, IoT, documents — all in separate systems with no connection.

⚠️

Lack of overview

No unified picture of building condition and assets.

📄

Manual inventory

Time-consuming, error-prone and rarely kept up to date.

🌐

Poor accessibility

Information requires specialized tools and experts to access.

The Solution

Geminus unifies all building data into a digital twin — accessible for everyone, everywhere.

🧊

xeokit-based viewer with IFC models

📸

360° Panorama

NavVis Ivion integration with room navigation

🧠

AI Assistants

Ask Gunnar about data, Ilean about documents

⚡

IoT & Sensors

Real-time data from Senslinc sensors

3D + 360° Viewer

Explore buildings in 3D models and 360° panoramas — side by side or as a Virtual Twin.

360° Integration

NavVis Ivion SDK, camera sync 3D↔360°, POI markers

🏢

View Modes

🖥️

AI Asset Detection

Automated inventory from 360° panoramas — powered by Google Gemini.

1

Configure

Select building, templates, and image source.

2

Scan

AI analyzes panoramas and identifies objects.

3

Review

Detections with confidence scores in a review queue.

4

Register

Auto-created as assets with POI in Ivion.

🔍

AI Assistants

Ask about the building in natural language — or use voice commands.

💬 Gunnar

Data assistant — assets, rooms, areas, work orders.

"How many fire extinguishers on floor 3?"

📄 Ilean

Document assistant — drawings, manuals, instructions.

"Show the ventilation diagram for fan room B2."

🎤 Voice Control

Web Speech API for hands-free navigation and queries.

🤖

Mobile & QR

Fully responsive — with specialized flows for field workers.

📱 QR Fault Reporting

Scan QR on an asset → report directly on mobile. Photo, describe, submit.

🗂️ Mobile Inventory

Step-by-step wizard with GPS, category selection and quick registration.

👁️ Mobile Viewers

Full-screen 3D and 360° viewers with touch gestures.

📱

Technology & Architecture

Built with modern web technologies and cloud services — no server needed.

Frontend

React 18 + TypeScript
Vite — fast development & HMR
Tailwind CSS + shadcn/ui
xeokit SDK — WebGL BIM rendering
NavVis Ivion SDK — 360° panorama
Mapbox GL — map with clusters

Backend & Integrations

Lovable Cloud — database, auth, storage
Edge Functions — Deno serverless
Asset+ FM API — asset & facility data
NavVis Ivion API — POI & panorama
Senslinc API — IoT sensor data
Google Gemini — AI vision & chat

⚡ Fully serverless — built and hosted on Lovable
